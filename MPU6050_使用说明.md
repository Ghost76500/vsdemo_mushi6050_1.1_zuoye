# MPU6050 硬件I2C使用说明

## 概述
本项目已将MPU6050从软件IIC改为使用STM32F103C8T6的硬件I2C2接口（HAL库），并添加了pitch、roll、yaw三轴姿态角读取功能。

## 硬件连接
- **I2C2_SCL**: PB10
- **I2C2_SDA**: PB11
- **MPU6050_AD0**: 接GND (I2C地址为0x68)

## 主要修改内容

### 1. I2C配置 (`Core/Src/i2c.c`)
- 添加了I2C2初始化函数 `MX_I2C2_Init()`
- 配置I2C2时钟速度为400kHz
- GPIO配置为开漏输出模式

### 2. MPU6050驱动 (`User/mpu6050.c` 和 `User/mpu6050.h`)
- 移除了软件IIC相关代码
- 使用HAL库函数：
  - `HAL_I2C_Mem_Write()` - 写寄存器
  - `HAL_I2C_Mem_Read()` - 读寄存器
- 添加姿态角计算功能

### 3. 新增姿态角读取API

#### 数据结构
```c
typedef struct
{
    float pitch;  // 俯仰角（度）
    float roll;   // 横滚角（度）
    float yaw;    // 航向角（度）
} MPU6050_Attitude;
```

#### API函数

##### 1. 获取姿态角（结构体方式）
```c
uint8_t MPU_Get_Attitude(MPU6050_Attitude *attitude);
```
- **参数**: `attitude` - 指向MPU6050_Attitude结构体的指针
- **返回值**: 0-成功，其他-失败
- **示例**:
```c
MPU6050_Attitude attitude;
if(MPU_Get_Attitude(&attitude) == 0)
{
    printf("Pitch: %.2f, Roll: %.2f, Yaw: %.2f\n", 
           attitude.pitch, attitude.roll, attitude.yaw);
}
```

##### 2. 获取姿态角（指针方式）
```c
void MPU_Get_Pitch_Roll_Yaw(float *pitch, float *roll, float *yaw);
```
- **参数**: 
  - `pitch` - 俯仰角指针
  - `roll` - 横滚角指针
  - `yaw` - 航向角指针
- **示例**:
```c
float pitch, roll, yaw;
MPU_Get_Pitch_Roll_Yaw(&pitch, &roll, &yaw);
printf("Pitch: %.2f\n", pitch);
printf("Roll: %.2f\n", roll);
printf("Yaw: %.2f\n", yaw);
```

## 姿态角计算原理

### 传感器配置
- 加速度计量程: ±2g (灵敏度: 16384 LSB/g)
- 陀螺仪量程: ±2000°/s (灵敏度: 16.4 LSB/(°/s))
- 采样率: 100Hz

### 互补滤波算法
系统采用互补滤波器融合加速度计和陀螺仪数据：

1. **加速度计**: 提供静态角度（抗漂移）
   - Pitch计算: `atan2(ay, sqrt(ax² + az²))`
   - Roll计算: `atan2(-ax, az)`

2. **陀螺仪**: 提供动态角速度积分（快速响应）
   - 角度积分: `angle += gyro * dt`

3. **融合公式**:
   ```
   attitude = 0.98 × gyro_angle + 0.02 × accel_angle
   ```
   - 98%权重给陀螺仪（保持快速响应）
   - 2%权重给加速度计（长期稳定性）

### 注意事项
- **Yaw角**: 仅通过陀螺仪积分计算，存在累积漂移。如需精确航向角，需添加磁力计(如HMC5883L)
- **初始化**: 首次调用`MPU_Get_Attitude()`时会初始化时间戳，建议系统启动后先读取几次数据以稳定输出

## 主函数使用示例 (`Core/Src/main.c`)

```c
#include "main.h"
#include "i2c.h"
#include "mpu6050.h"
#include <stdio.h>

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    
    // 初始化外设
    MX_GPIO_Init();
    MX_I2C2_Init();  // 必须初始化I2C2
    
    // 初始化MPU6050
    if(MPU_Init() == 0)
    {
        // 初始化成功
    }
    else
    {
        // 初始化失败，检查硬件连接
        Error_Handler();
    }
    
    float pitch, roll, yaw;
    
    while(1)
    {
        // 读取姿态角
        MPU_Get_Pitch_Roll_Yaw(&pitch, &roll, &yaw);
        
        // 显示或处理数据
        printf("Pitch: %.2f°, Roll: %.2f°, Yaw: %.2f°\n", 
               pitch, roll, yaw);
        
        HAL_Delay(100);  // 100ms更新一次
    }
}
```

## 与OLED显示配合使用

```c
#include "OLED.h"

char oled_buf[32];
float pitch, roll, yaw;

while(1)
{
    MPU_Get_Pitch_Roll_Yaw(&pitch, &roll, &yaw);
    
    OLED_Clear();
    
    // 显示Pitch

    sprintf(oled_buf, "%.2f", pitch);
    OLED_ShowString(48, 0, oled_buf, OLED_8X16);
    
    // 显示Roll
    OLED_ShowString(0, 2, "Roll:", OLED_8X16);

    // 显示Yaw
    OLED_ShowString(0, 4, "Yaw:", OLED_8X16);
    sprintf(oled_buf, "%.2f", yaw);
    OLED_ShowString(48, 4, oled_buf, OLED_8X16);
    
    OLED_Update();
    HAL_Delay(100);
}
```

## 原有API保持不变

以下函数仍可正常使用：
```c
uint8_t MPU_Init(void);                                      // 初始化
uint8_t MPU_Get_Gyroscope(short *gx, short *gy, short *gz); // 读陀螺仪原始值
uint8_t MPU_Get_Accelerometer(short *ax, short *ay, short *az); // 读加速度原始值
short MPU_Get_Temperature(void);                            // 读温度值(×100)
```

## 调试建议

1. **检查I2C连接**: 使用逻辑分析仪或示波器检查SCL/SDA信号
2. **验证器件ID**: `MPU_Read_Byte(MPU_DEVICE_ID_REG)` 应返回 0x68
3. **检查时钟配置**: 确保I2C2时钟已使能
4. **上拉电阻**: SCL和SDA需要外部4.7kΩ上拉电阻（如果模块未集成）

## 性能参数

- **I2C速度**: 400kHz (Fast Mode)
- **数据更新率**: 建议100ms (10Hz)
- **角度精度**: ±2° (静止状态)
- **响应时间**: <50ms

## 常见问题

### Q1: 初始化失败
- 检查I2C2硬件连接
- 确认MPU6050的AD0引脚接地
- 检查供电是否正常（3.3V）

### Q2: 角度漂移
- Yaw角会累积漂移（需要磁力计校正）
- Pitch/Roll漂移需检查传感器是否稳定放置

### Q3: 读取数据全为0
- 检查I2C2是否正确初始化
- 确认hi2c2句柄已声明为extern
